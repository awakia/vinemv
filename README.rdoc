Vine MV
====

曲名を入力すると、Vineの動画を使ったミュージックビデオを自動生成するWebアプリケーションです。

If you type a music title, This app auto-create a music video using Vine movies.

## Demo
Herokuで確認できます。

You can check this app on Heroku.

+ <https://vinemv.herokuapp.com/>

## Requirement
アプリケーションは以下の環境での動作確認を行いました。

This app is guaranteed to run only the following environment.

+ Ruby 2.2.2p95
+ Sinatra 1.4.6

## Usage
+ **class/extract.rb**の鍵を書き換える。Change Keys at **class/extract.rb**.
+ ``bundle install``
+ ``ruby main.rb``
